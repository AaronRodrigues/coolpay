# CoolPay

### Installation

1. Clone this repository by typing the following in the command line.
   ```
   git@github.com:AaronRodrigues/coolpay.git
   ```
2. To install dependencies, type:
   ```
   bundle install
   ```
3. Create a .env file in the root directory
4. Fill it with 
   ```
   USERNAME = username
   API_KEY  = apikey
   ```
